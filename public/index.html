API Basejump: Timestamp microservice

User Stores:

  1. I can pass a string as a parameter, and it will check to see whether it is a unix timestamp or a natural language date(example: January 1, 2016)
  2. If it contains a timestamp it will return both the natural language and unix timestamps
  3. If it does not it will return null for both properties

Example usage:

https://timestamp-microservice-thespitefuloctopus.c9users.io/December%2015,%202015
https://timestamp-microservice-thespitefuloctopus.c9users.io/1450137600

Example output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
